# 🎵 Robotic Musical Glove

A wearable system that turns hand gestures into music using an ESP32-S3. The glove sends MIDI signals to digital audio workstations such as GarageBand, allowing users to play instruments in real time through natural hand movement.

---

## 🚀 Overview

The Robotic Musical Glove makes music creation more intuitive by mapping finger input and gestures to musical notes. It combines embedded systems, real time processing, and MIDI communication to create an interactive performance tool.

---

## ✨ Features

* 🎹 Gesture based note triggering
* 🔌 MIDI output over USB with plug and play support
* ⚡ Low latency real time performance
* 🧩 Flexible mapping for different instruments
* 🎧 Works with GarageBand and other MIDI compatible software

---

## 🛠️ Hardware

* ESP32-S3 microcontroller
* Flex sensors or buttons for finger input
* USB-C connection for MIDI communication
* Supporting components such as resistors and wiring

---

## 💻 Software

* Arduino IDE
* ESP32 board support package
* USB MIDI implementation

---

## ▶️ Setup

1. Install Arduino IDE
2. Install ESP32 board support
3. Open `Musical-Glove.ino`
4. Select ESP32-S3 as the target board
5. Upload the code to the device
6. Connect the board to your computer via USB
7. Open a DAW and select the device as a MIDI input

---

## 🎮 Usage

* Trigger notes using finger movements or button inputs
* Map inputs to different instruments in your DAW
* Use for live playing, experimentation, or recording
